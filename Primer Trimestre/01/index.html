HTML Y CSS

HTML

Es un lenguaje de marcado usado para estructurar el contenido de una página web.

Algunas de sus principales funciones/características son: definir títulos, esto esta en negrita párrafos, listas, tablas, imágnees, vídeos, formularios, etc.; introducir goolge el uso de etiquetas semánticas (p.ej.: header, nav, article, footer); tener soporte de audio y vídeo de forma nativa; permitir trabajar con gráficos (etiquetas canvas y svg). ¡Iremos descubriendo clase a clase todo lo que nos ofrece!

CSS

Es un lenguaje que define la presentación y el diseño de las páginas HTML.

Algunas de las funciones principales son: cambiar colores, fuentes márgenes, tamaños, posiciones; crear diseños responisvos (que se adapten a móviles, tablets, pantallas, PC, etc.); añadir animaciones y transiciones.
